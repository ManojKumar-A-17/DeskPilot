# ThinkAuto - AI-Powered IT Helpdesk

## Project Overview

ThinkAuto uses AI to automate ticket routing, predict issues, and accelerate resolution — so your team can focus on what matters.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

Follow these steps to set up the project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd thinkauto_frontend

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm test` - Run tests
- `npm run test:watch` - Run tests in watch mode

## Features

- **Smart Routing**: AI automatically assigns tickets to the right technician
- **SLA Tracking**: Real-time SLA monitoring with automatic escalation
- **Analytics**: Deep insights into your support operations
- **AI Assistant**: Intelligent chatbot for instant help
- **Multi-role Support**: Employee, Technician, and Admin dashboards
- **Knowledge Base**: Searchable documentation and solutions
